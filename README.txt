## Installation and Setup

To clone and view this website locally:

1. Clone the repository:
git clone https://github.com/mingminggl/mingminggl.github.io.git

2. Navigate to the project directory:
cd mingminggl.github.io

3. Open `index.html` in your web browser.

Since the website is developed in HTML, there's no need for additional setup or server-side components for basic viewing.

## Usage

The website is structured into several sections, including:

- **Home**: A brief introduction to who I am and what I do.
- **Portfolio**: A showcase of my past projects and works.
- **About Me**: More detailed information about my background and skills.
- **Contact**: How to reach me for potential opportunities or inquiries.

Feel free to browse through the sections to know more about my professional journey.
